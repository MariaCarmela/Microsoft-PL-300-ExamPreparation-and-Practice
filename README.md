## 📌 Description

This repository collects the **exercises**, **notes**, and **screenshots** produced while studying for the **Microsoft PL-300 – Power BI Data Analyst** certification.

The material follows the official course offered by Microsoft on Coursera:

> 🎓 [Microsoft PL-300 Exam Preparation and Practice](https://www.coursera.org/learn/microsoft-pl-300-exam-preparation-and-practice/) — *by Microsoft on Coursera*
>
> **Level:** Beginner | **Duration:** ~4 weeks (10 hours/week) | **Assignments:** 16

The goal is to document the learning journey, consolidate key concepts, and prepare in a structured way for the certification exam.

## 🏆 Certification Goal

| Field | Detail |
|-------|--------|
| **Exam** | PL-300 – Microsoft Power BI Data Analyst |
| **Certification** | Microsoft Certified: Power BI Data Analyst Associate |
| **Cost** | ~€165 (indicative price, may vary) |
| **Exam language** | Available in English and other languages |
| **Renewal** | Free annual renewal via Microsoft Learn, without retaking the paid exam |
| **Official link** | [PL-300 Exam Page](https://learn.microsoft.com/en-us/certifications/exams/pl-300/) |

> 💡 Certification renewal takes place annually through a **free online assessment** on Microsoft Learn, without having to take the full exam again.
